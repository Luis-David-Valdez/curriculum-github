Carro seguidor de línea

El primer proyecto que desarrollamos consistió en la construcción de un carro seguidor de línea. Para su funcionamiento utilizamos dos sensores infrarrojos TCRT5000, los cuales permitieron detectar el contraste entre la línea guía y la superficie. El chasis del vehículo fue fabricado con material acrílico, mientras que las llantas fueron diseñadas e impresas en 3D para optimizar el agarre y la estabilidad del carro en movimiento.

Como unidad de procesamiento utilizamos una placa ESP32, que funcionó como microcontrolador central para recibir las señales de los sensores y controlar el movimiento del vehículo. El sistema fue alimentado mediante baterías recargables, lo que permitió su operación autónoma.

Este proyecto nos permitió reforzar conocimientos en electrónica, programación y diseño mecánico, además de fomentar el trabajo en equipo y la resolución de problemas en tiempo real.

Reloj con PIC y contador binario

En este proyecto desarrollé un reloj utilizando un microcontrolador PIC. El sistema estaba programado para contar un minuto de forma precisa, funcionando como un temporizador básico. Además, se incorporó una función adicional: un contador en binario, el cual se controlaba mediante un DIP switch.

Este componente permitía al usuario seleccionar diferentes combinaciones de bits, las cuales eran interpretadas por el microcontrolador para activar el conteo correspondiente en binario. La implementación de esta función ayudó a comprender de forma práctica el funcionamiento de los sistemas digitales y la lógica binaria.

Este proyecto fortaleció mis habilidades en programación de microcontroladores, uso de entradas digitales y comprensión del sistema binario, además de mejorar mi capacidad para integrar hardware con software de forma funcional.

Instalación de Cámaras de Seguridad con DVR

Este proyecto se realizó durante el periodo de estancias profesionales y consistió en la instalación de un sistema de videovigilancia que incluía cámaras de seguridad y un dispositivo DVR (Digital Video Recorder) para el monitoreo y almacenamiento de video.

Fue el proyecto final con el que concluimos nuestras estancias, y se llevó a cabo en colaboración con dos compañeros más. Durante su ejecución, aplicamos de manera práctica todos los conocimientos y habilidades adquiridas a lo largo de las capacitaciones previas, tales como instalación física de equipos, configuración de sistemas de videovigilancia y manejo del DVR.

Este proyecto no solo reforzó nuestras competencias técnicas, sino también el trabajo en equipo, la responsabilidad profesional y la atención al detalle en la ejecución de tareas reales en un entorno laboral.
